    1. Cicerone

      November 2022 - February 2023

      The full-stack Flutter application developed for this project combines various technologies to provide seamless navigation within an indoor museum. Firebase serves as the database, enabling efficient data storage and retrieval, as well as supporting CRUD operations for managing the museum's exhibits and additional information. The application incorporates a custom indoor museum map designed in RIVE, which offers an interactive and visually appealing interface. To ensure optimal route planning, Dijkstra's algorithm is implemented, allowing users to find the shortest path between any two locations within the museum. Additionally, the application can be easily downloaded by scanning a QR code, making it convenient for users to access and explore the museum with just a simple scan. This application represents a powerful tool for enhancing the visitor experience, providing them with a user-friendly and efficient way to explore, manage, and navigate through the museum's exhibits.
